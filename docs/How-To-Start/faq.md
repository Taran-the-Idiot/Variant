# FAQ

## What is Hack Club?

Hack Club is a 501(c)(3) non-profit organisation that has set its mission to helping students learn how to code, engineer and take up stem as both a hobby and a career pathway. They do this through programs where they reward teens for making projects. Variant is just one of the many programs run by Hack Club.


## What is Variant?

Variant is a YSWS(you ship, we ship) program where you make a hardware project that has a weird gimmick to it, and we send you funding to make the project + a cool merch pack
